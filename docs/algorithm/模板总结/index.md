> 无论学习什么，都要把体系列出来，然后再分解（庖丁解牛），重复练习单独的部分。遇到单个的知识点，一定要再插入到自己的知识体系中。这个才是符合人脑的记忆结构，比如和已有知识产生链接，才能被更好的记忆和存储。链接得越多，记忆提取就会越准备，越快。
